## Description

A Multipage E-Commerce website made with JavaScript, showcases the products the website is selling, testimonial cards from previous buyers and the benefits of owning the products. Contains a Home page, shopping page, products page, carts page and a confirmation page which simulates buying a product.

# Tech Used
- HTML
- CSS
- JavaScript

# Features

- Hover animation over the product cards
- Toast notification when product is added to cart
- Functioning Cart and Checkout 
- Zoom in on Product Image in Product page
- Image carousel for product Images
- Search Engine
- Filter and Sort fucntion
- Dynamically rendered product cards from a JavaScript data structure
